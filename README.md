## Setup

### Frontend

#### Requirements
- `Node`

#### Getting Started

1. `cd frontend`
2. `npm i`
3. `npm start`

### Backend

#### Getting Started
1. `cd backend`
2. `python3 -m pip install -r requirements.txt`
3. `python3 routes.py`
